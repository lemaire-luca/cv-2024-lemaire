Mes compétences

Toutes ses compétences ont été développées et entretenues tout au long de mon parcours dans un bachelier en infographie. Au fil de cette formation, j’ai perfectionné mes compétences dans le domaine.

Découvrez-moi

HTML

HTML (HyperText Markup Language) : Découvert lors de ma première année de bachelier en web, HTML est le langage de balisage utilisé pour structurer les pages web.

PHP

PHP (Hypertext Preprocessor) : Au cours de ma deuxième année de bachelier en web, j’ai exploré PHP, un langage de script côté serveur utilisé pour le développement d’applications web dynamiques, notamment l’interaction avec les bases de données.

CSS

CSS (Cascading Style Sheets) : Au cours de ma première année de bachelier en web, j’ai exploré CSS, le langage de feuilles de style qui permet de styliser et mettre en page le contenu HTML.

JAVASCRIPT

JavaScript : Découvert au cours de ma deuxième année de bachelier en web, JavaScript s’est révélé être un langage de programmation côté client essentiel pour rendre les pages web interactives et dynamiques.

PHOTOSHOP

Durant ma première année de bachelier en web, j’ai appris à utiliser Photoshop, un logiciel de retouche d’images et de création graphique, particulièrement utile pour l’édition de photos et la conception visuelle.

ILLUSTRATOR

Au cours de ma première année en web, j’ai exploré Illustrator, un logiciel de création graphique vectorielle essentiel pour concevoir des logos, des illustrations et des éléments graphiques.

MAYA

Découvert en première année de bachelier, Maya est un logiciel de modélisation 3D utilisé pour créer des animations, des effets visuels et des jeux.
